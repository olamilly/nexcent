# Nexcent
Beautiful landing page for a business.
https://nexcent-pi-green.vercel.app/

## Project setup

```sh
git clone https://github.com/olamilly/nexcent.git
```

```sh
cd nexcent
```

```sh
npm install
```

### Compile for development
```sh
npm run serve
```

### Build for production
```sh
npm run build
```
